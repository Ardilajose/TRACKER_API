## 🎯 Objetivos

### Objetivo General
Desarrollar una aplicación web que permita gestionar y analizar ingresos y gastos personales de forma sencilla y eficiente.

### Objetivos Específicos
- ✅ Diseñar una estructura de datos para almacenar información financiera
- ✅ Implementar funcionalidades para registrar ingresos y gastos
- ✅ Clasificar movimientos por categorías
- ✅ Calcular el balance total automáticamente
- ✅ Mostrar información resumida al usuario

---

## 📊 Funcionalidades

| Función | Descripción | Estado |
|---------|-------------|--------|
| Registrar ingresos | Agregar transacciones de ingreso | ✅ |
| Registrar gastos | Agregar transacciones de gasto | ✅ |
| Clasificar transacciones | Categorías: comida, transporte, ocio, ingreso | ✅ |
| Listar transacciones | Ver todas las transacciones registradas | ✅ |
| Eliminar registros | Eliminar transacciones no deseadas | ✅ |
| Calcular balance | Total ingresos - total gastos | ✅ |
| Resumen financiero | Visualización de totales y categorías | ✅ |
| Modo oscuro/claro | Alternar entre temas visuales | ✅ |
| Persistencia de datos | Almacenamiento en archivo JSON | ✅ |
